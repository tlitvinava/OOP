# Canvas Drawing Application

## Description
Paint is a versatile graphical tool that allows users to create and manipulate geometric shapes on a digital canvas. Users can draw rectangles, circles, and triangles, move them around, remove them, and undo or redo their actions. The application also supports customizable symbol and background colors, enhancing the user experience. Designed for educational purposes, this application provides a solid foundation for understanding object-oriented programming and graphical user interfaces in Python.

## Features
- Create a canvas of specified dimensions.
- Draw geometric shapes: rectangles, circles, and triangles.
- Move and remove shapes using their unique identifiers.
- Undo and redo actions for flexible editing.
- Change symbol and background colors.
- Save and load canvas states from files.
